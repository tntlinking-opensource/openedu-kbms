# 项目介绍

OpenEDU知识库系统是一款基于LLM大模型的信息平台，提供开箱即用的数据处理、模型调用、本地知识库构建、智能体管理等能力，可轻松打造专属的AI应用。本平台特别适用于中高职院校场景，例如构建课程本地知识库、学生在线学习助理、学校招生咨询、办事服务等智能客服问答场景，有效解决教育机构AI落地面临的技术门槛高、部署成本高、迭代周期长等问题。

如需要其他产品源码，可访问天天开源官网。
